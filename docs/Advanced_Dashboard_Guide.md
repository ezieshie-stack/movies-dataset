## PHASE 1: Build Individual Sheets (8 Sheets + 3 Sparklines)

### Sheet 1: KPI â€” Genre Revenue Card (Drama)

**Goal:** Colored card with big revenue number + % change label

1. Create **New Worksheet** â†’ Name: `KPI Drama Revenue`
2. Add a **Filter**: Drag `Primary Genre` â†’ Select only "Drama"
3. Drag `Revenue` to **Text** â†’ Aggregation: **SUM**

### Sheet 3: Top Movies by Revenue â€” Ranked List

**Goal:** Numbered list with inline colored bars (like "Top Products by Revenue")

### Sheet 4: Avg Rating Gauge

**Goal:** Speedometer-style gauge showing average IMDB rating

**Method:** Use a pie chart trick to create a semicircular gauge

1. Create **New Worksheet** â†’ Name: `Rating Gauge`
2. Create **Calculated Fields:**

   **Gauge Value** (the metric, normalized to 180Â°):
   ```
   AVG([IMDB Score]) / 10
   ```

   **Gauge Remaining:**
   ```
   1 - [Gauge Value]
   ```

   **Gauge Background** (bottom half, always 1 = 180Â°):
   ```
   1
   ```

3. Create a **Pie Chart:**
   - Drag `Gauge Value`, `Gauge Remaining`, `Gauge Background` to **Angle** (as measure values)
   - Change Mark type to **Pie**
   - Assign colors:
     - Gauge Value â†’ Gradient (Red â†’ Yellow â†’ Green based on value)
     - Gauge Remaining â†’ Light gray
     - Gauge Background â†’ White (hides the bottom half)

4. **Simpler Alternative â€” Use a BAN (Big Ass Number):**

### Sheet 5: Rating Breakdown Bars

**Goal:** Horizontal bars showing count of movies per rating range (like star breakdown)

1. Create **New Worksheet** â†’ Name: `Rating Breakdown`
2. Create **Calculated Field**: `Rating Tier`
   ```
   IF [IMDB Score] >= 8 THEN "â­â­â­â­â­ (8-10)"
   ELSEIF [IMDB Score] >= 6 THEN "â­â­â­â­ (6-8)"
   ELSEIF [IMDB Score] >= 4 THEN "â­â­â­ (4-6)"
   ELSEIF [IMDB Score] >= 2 THEN "â­â­ (2-4)"
   ELSE "â­ (0-2)"
   END
   ```
3. Drag `Rating Tier` to **Rows**
4. Drag `Title` (COUNT) to **Columns**
5. Add **Label**: Percentage of total
   ```
   SUM([Number of Records]) / TOTAL(SUM([Number of Records]))
   ```
   Format as percentage
6. Color bars with gradient (green for high ratings, red for low)
7. Sort: 5-star at top, 1-star at bottom
8. **Clean up**: Remove axes, add value labels at bar end

### Sheet 6: Budget Distribution Doughnut

**Goal:** Doughnut chart showing budget category split

1. Create **New Worksheet** â†’ Name: `Budget Doughnut`
2. Change Mark type to **Pie**
3. Drag `Budget Category` to **Color**
4. Drag `Number of Records` to **Angle**
5. **Make it a doughnut (Dual-Axis method):**
   - Put `MIN(1)` on **Rows** twice â†’ Right-click second â†’ **Dual Axis** â†’ **Synchronize Axes**
   - First axis: Full pie (with Budget Category colors)
   - Second axis: Smaller white circle (remove all fields, set color to white, increase size)
   - Hide both axes
6. Add **percentage labels** on the first pie:
   ```
   SUM([Number of Records]) / TOTAL(SUM([Number of Records]))
   ```
7. Color palette: Use teal, green, yellow, orange for categories

### Sheet 8: Movie Details Table

**Goal:** Clean list with colored profitability status pills

1. Create **New Worksheet** â†’ Name: `Movie Details`
2. Drag to **Rows**: `Title`, `Primary Genre`, `Release Year`, `Budget`, `Revenue`
3. Create **Calculated Field**: `Profitability Status`
   ```
   IF [Is Profitable] = 1 THEN "Profitable"
   ELSEIF [Revenue] > 0 THEN "Not Profitable"
   ELSE "No Data"
   END
   ```
4. Create **Calculated Field**: `ROI Display`
   ```
   STR(ROUND([ROI], 1)) + "%"
   ```
5. **For colored status pills:**
   - Put `Profitability Status` on **Color**
   - Assign: Green (#27AE60) = Profitable, Red (#E74C3C) = Not Profitable, Gray (#CCCCCC) = No Data
   - Mark type: **Circle** or **Square**
   - Place profitability as the last column
6. **Add colored circle icons** (like Order ID circles):
   - Create Calculated Field: `Status Icon Color`
     ```
     IF [ROI] > 300 THEN "Excellent"
     ELSEIF [ROI] > 100 THEN "Good"
     ELSEIF [Is Profitable] = 1 THEN "OK"
     ELSE "Poor"
     END
     ```
   - Use as a Shape mark with custom colored circles
7. **Clean formatting:**
   - Remove gridlines
   - Add alternating row shading (very light gray)
   - Set font to 10pt
   - Add "Show Filter" button as floating text object later
